# Deployer
Deployer is a [Create](https://www.curseforge.com/minecraft/mc-mods/create) library-addon used by me and our team to develop create addons.
Our aim is to provide a **Strong**, **Stable** and **Simple** library for registering features.
For our mods, **Blueprint** is packaged via **Jar-in-Jar** directly in the mods,
so no need to download it from any third party website.
Below, you will find information about installation and features.

## Features
- Registering custom factory gauges
- Creating an `AllIcons` child without mixing with the usage of [AllIconsHelper](src/main/java/net/liukrast/deployer/lib/util/AllIconsHelper.java)
- Help to create custom gauge ponder scenes via [GaugeHelper](src/main/java/net/liukrast/deployer/lib/util/ponder/GaugeHelper.java)
